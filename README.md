# **名稱:距離測試**  

**時間:2020/3/6~2020/3/13**

**內容:**  
透過超音波感測器測量物體距離  
並將其顯示在LCD及監控視窗上  

**補充:**  
監控視窗程式:若越靠近物體LED燈閃爍速度加快  
LCD程式:若距離超過100公分輸出error
